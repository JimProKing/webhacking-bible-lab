# problems/ — 문제별 폴더 (이곳이 메인입니다)

이 폴더가 **이 실습 교재의 핵심**입니다.

각 하위 폴더 = 하나의 문제.

`01-...` 폴더부터 순서대로 열어서 작업하세요.

한 폴더를 완전히 이해하고 풀이까지 읽으면 그 문제가 끝난 것입니다.

## 진행 방법 (추천)

1. `ls` 또는 파일 탐색기로 problems/ 폴더를 봅니다.
2. 가장 작은 번호의 아직 안 한 폴더를 엽니다.
3. 그 폴더 안의 `README.md` 를 읽습니다.
4. 미션을 수행합니다.
5. `풀이.md` 를 읽고 "아 이게 근본 원인이구나"를 깨닫습니다.
6. 다음 폴더로 이동.

이 방식이 "내가 뭘 끝냈고 뭘 해야 하는지"를 가장 명확하게 보여줍니다.

## 전체 커리큘럼 (간단)

- 01~03 : 기초 + HTTP + Burp 근육 (Ch01~02)
- 04~08 : SQL Injection 완전 정복 (Ch04)
- 09~13 : 주요 공격 기법 (Ch05~11)
- 14~20 : 방어 실습 + 추가 Recon (Ch12~19 + Ch03)
