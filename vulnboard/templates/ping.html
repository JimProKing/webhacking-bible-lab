💡 Command Injection 연습 방법 (Windows 기준)

이 페이지는 ping -n 1 {host} 형태로 명령어를 만들고 shell=True 로 실행합니다.

시도할 페이로드
  • 127.0.0.1 & whoami — 명령어 연결 (&)
  • 127.0.0.1 | dir — 파이프
  • 127.0.0.1 && type vulnboard.db — DB 파일 내용 읽어보기 (가능할까요?)
  • 127.0.0.1 & net user — 사용자 목록 보기
학습 포인트: 사용자가 입력한 값을 검증 없이 쉘에 넘기면, 공격자는 &, |, ;, && 등을 이용해 원하는 명령어를 추가로 실행할 수 있습니다.
실제 방어는 입력값 검증 + escaping + 가능하면 shell=False + 화이트리스트 방식으로 합니다.
